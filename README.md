# Caching Proxy Server

A CLI-based caching proxy server built using Node.js and Express.

## 🚀 Goal
This project will:
- Forward client requests to an origin server
- Cache responses
- Return cached responses for repeated requests

## 📁 Project Structure
caching-proxy/
│── src/
│   └── server.js
│── package.json
│── README.md

## ⚙️ Setup

Install dependencies:
npm install
